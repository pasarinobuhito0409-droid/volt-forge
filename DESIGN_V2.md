## D6 判断ドリル24問（別紙・全文このまま実装）

tag: 配線(wire)/保護(protect)/容量(power)/制御(ctrl)/接地(gnd)/安全(safety)
1. wire1: モーター動力線と微小センサ信号線を同じダクトに並走させた → **NG**「動力のノイズが信号に乗り移る（誘導）」→「対策: 別ダクト・距離を取る・交差は直角・シールド線」
2. wire2: 電線サイズを負荷電流と許容電流表から選定した → **OK**「電線は細いと発熱して燃える」→「長距離なら電圧降下の計算も追加」
3. wire3: 端子台の1端子に電線を3本差した → **NG**「端子の許容は通常2本まで。緩み・接触不良のもと」→「対策: 渡り端子・中継端子台を使う」
4. wire4: 長い配線路で電圧降下を計算した → **OK**「遠い機器ほど電圧が痩せる」→「24V機器は特にシビア。線を太くするか電源を近くに」
5. protect1: ブレーカー容量を電線の許容電流より大きくした → **NG**「ブレーカーより先に電線が燃える」→「対策: ブレーカーは電線を守るのが本務。電線基準で選ぶ」
6. protect2: モーターにサーマルリレー（過負荷でモーターを守る装置）を付けた → **OK**「拘束・過負荷の焼損防止の定石」→「設定値はモーター定格電流に合わせる」
7. protect3: ヒューズがよく切れるので容量を上げた → **NG**「ヒューズは身代わり。上げたら本体が燃える」→「対策: なぜ切れるかの原因調査が先」
8. protect4: 動力回路と制御回路の保護を分けた → **OK**「制御が生きていれば異常表示・安全停止ができる」→「一蓮托生の回路は原因究明も難しい」
9. power1: 24V電源をセンサ・電磁弁の起動突入電流を無視して選定した → **NG**「起動の一瞬は定格の数倍。電源が落ちて全滅」→「対策: 突入込みで余裕率を持つ」
10. power2: モーターの起動電流（定格の数倍）を考慮してブレーカーを選定した → **OK**「起動でトリップしない・でも異常では切れる絶妙な選定」→「モーターブレーカーという専用品もある」
11. power3: 将来の増設分の予備容量・予備端子を見込んだ → **OK**「盤の増設は必ず来る」→「予備20%は先輩の知恵」
12. power4: 発熱計算をせず密閉盤に機器を詰め込んだ → **NG**「盤内温度が上がると機器寿命が激減・誤動作」→「対策: 発熱合計を計算し、ファン・ルーバー・盤サイズで逃がす」
13. ctrl1: 非常停止をPLCのプログラムだけで処理した → **NG**「プログラムは凍る・バグる。非常停止が効かない機械は凶器」→「対策: 非常停止はハード回路で直接動力を切る」
14. ctrl2: センサの接点を「断線したら停止する」向き（安全側）に選んだ → **OK**「フェールセーフの基本。壊れ方まで設計する」→「b接点の使いどころ」
15. ctrl3: リレー接点の容量を負荷の突入電流で選んだ → **OK**「接点は突入で溶着する」→「ランプ・モーター・コンデンサ負荷は特に注意」
16. ctrl4: 電磁弁（コイル負荷）をサージ対策なしでPLC出力に直結した → **NG**「切った瞬間の逆起電力（サージ）で出力が死ぬ」→「対策: ダイオード・バリスタ・CR回路を並列に」
17. gnd1: 盤のアース線を塗装面の上からネジ留めした → **NG**「塗膜は絶縁体。アースが効いていない」→「対策: 塗装レス面か塗膜を剥がす＋菊座金」
18. gnd2: シールド線のシールドを何も考えず両端接地した → **NG**「両端の電位差で電流ループ＝逆にノイズ源」→「対策: 基本は片端接地。ケースバイケースを知った上で」
19. gnd3: 動力系と信号系のアースを1点で合流させた → **OK**「一点接地はノイズの基本作法」→「アースの経路も配線設計のうち」
20. gnd4: 原因不明の誤動作で、まず接地と配線経路を疑った → **OK**「ノイズトラブルの現場感覚」→「動力線との並走・アース浮きが二大容疑者」
21. safety1: 盤の扉を開けると動力が切れるインターロックを付けた → **OK**「充電部に触れる前に電気を切る」→「保守する人の命を設計で守る」
22. safety2: 200Vの端子が露出したまま納品した → **NG**「感電・短絡事故」→「対策: 端子カバー・アクリルカバー・危険表示」
23. safety3: 盤内ポケットに回路図を入れた → **OK**「10年後の保全員への最強の贈り物」→「図面がない盤は開けるのも怖い」
24. safety4: 配線の色ルール（AC/DC/接地の色分け）を「動くからヨシ」で無視した → **NG**「次に触る人が誤認して感電・短絡する」→「対策: 色は情報。社内/JIS慣例に従う」
